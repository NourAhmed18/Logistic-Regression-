# Logistic-Regression-
# Breast Cancer Classification 🧬

## 📌 Description
This project applies Logistic Regression to classify breast cancer tumors as **Benign** or **Malignant** based on medical features.

---

## 🔍 Project Steps
- Data Loading & Cleaning
- Feature Selection (dropping irrelevant columns)
- Encoding target variable (M = 1, B = 0)
- Data Splitting (80% training, 20% testing)
- Feature Scaling using StandardScaler
- Model Training using Logistic Regression
- Model Evaluation

---

## 🤖 Model Used
- Logistic Regression

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Results
- The model achieved high accuracy in classifying tumors.
- The confusion matrix shows that most predictions were correct.
- Logistic Regression proved to be effective for binary classification.

---

## 📉 Evaluation Metrics
- Accuracy Score
- Confusion Matrix

---

## 🧠 Key Insights
- The model can effectively distinguish between benign and malignant tumors.
- Feature scaling improved model performance.
- Logistic Regression is simple, interpretable, and suitable for medical datasets.
